# steam_monitoring
Скрипт для мониторинга загрузки игр в Steam

## Требования
- Python 3.6 или выше
- Windows (для работы с реестром)
- Установленный Steam

## Файлы
- **steam_monitor.py** - основной скрипт (требует установленный Steam)
- **steam_monitor_demo.py** - демо версия с тестовыми данными (не требует Steam)

## Что делает
- Находит путь к Steam через реестр Windows
- Читает логи Steam для получения информации о загрузках
- Выводит название игры, скорость и статус
- Повторяет проверку каждую минуту в течение 5 минут

## Как запустить
```bash
python steam_monitor.py
